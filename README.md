# 概述

Socean是Solana区块链上的质押池。它的目标是：

1. 解锁Solana生态中已质押的SOL的应用价值
2. 提升用户的质押回报
3. 更加易于监控和管理质押
4. 以公平且透明的方式激励和奖励验证者
5. 在防范女巫攻击的同时，增强去中心化和抗审查性

Socean最终会演变为一个完全自主、由数据驱动的算法质押池。
